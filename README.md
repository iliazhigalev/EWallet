# EWallet

## Описание

EWallet - это приложение для обработки транзакций платёжной системы. Приложение реализует REST API для создания кошельков, перевода средств, получения истории транзакций и текущего состояния кошелька.

## Стек технологий

- Go 1.22.3
- PostgreSQL
- Docker
